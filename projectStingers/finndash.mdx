A dynamic dashboard for Finnhub.io's free financial API. Specify datasets, pivot through data, 
and collaborate using a GraphQL interface and excel templating.  

<b>Front End:</b> React, Typescript, Redux and is styled with MaterialUi.  

<b>Back End:</b> API created using Express. Account data is saved in Postgres. Data Caching is handled by MongoDB.

<b>Testing:</b> Testing are written with Testing Library and Jest is used as the test runner. Focus on integration tests. 

<b>Hosting & Deployment:</b> AWS EC2 using an Ubuntu server VM. Deployed using Docker Compose.
